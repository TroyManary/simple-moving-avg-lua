Class simpleMovingAvg

Simple Moving Average Filter

This file contains a lua class for a Simple Moving Average Filter as described here: http://en.wikipedia.org/wiki/Moving_average#Simple_moving_average

The class is instantiated (new) with the number of elements to include in the average.

Then the update function is called with each new value to input to the filter.
